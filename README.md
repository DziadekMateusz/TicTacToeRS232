# TicTacToeRS232
Tic-tac-toe game simulator written in C# using Windows Forms App (.NET Framework), using RS232 communication protocol to exchange information between 2 players. To play this minigame, a USB connection between two devices or virtual serial tool is required. Additionally, few other functions were implemented, such as synchronised trackbar, buttons with pre-written messages and textbox for sending custom messages, to further test the abovementioned communication protocol. Project designed and created during communication interfaces lessons at university.
